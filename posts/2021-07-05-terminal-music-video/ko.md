## 1\. ВъёВЮўВЮў ВЮ┤в»ИВДђвЦ╝ [вфевЁИьЂгвАг ВЮ┤в»ИВДђ](https://en.wikipedia.org/wiki/Binary_image)вАю в│ђьЎў

вфевЁИьЂгвАг ВЮ┤в»ИВДђвіћ 1в╣ёьіИ Ж╣іВЮ┤ВЮў ВЮ┤в»ИВДђвАю, `#000000`ВЮў ВЎёв▓йьЋю Ж▓ђВаЋЖ│╝ `#FFFFFF`ВЮў ВЎёв▓йьЋю ьЋўВќЉВю╝вАювДї ВЮ┤вБеВќ┤ВДђвіћ ВЮ┤в»ИВДђВЮ┤вІц.
ВЮ┤ ьЋўВќЉЖ│╝ Ж▓ђВаЋВЮё ВаљВъљВЌљВёю Ж░Ђ ВаљВЮ┤ В╣аьЋ┤ВДё (raised) ВЃЂьЃюВЎђ ЖиИваЄВДђ ВЋіВЮђ ВЃЂьЃюВЌљ вїђВЮЉВІюьѓг Вѕў ВъѕвІц.

ВаљВъљВЮў ЖИ░в│ИВаЂВЮИ ВЌГьЋаВЮђ **Ж▓йЖ│ёВёа**Ж│╝ **ьўЋВ▓┤**вЦ╝ ЖхгвХёьЋа Вѕў ВъѕвЈёвАЮ вЈёВЎђВБ╝віћ Ж▓ЃВЮ┤вІц.
ВЮ┤в»ИВДђвЦ╝ ьЮЉв░▒ВЮў ьўЋьЃювАю в│ђьЎўьЋю вњц 1в╣ёьіИ ьЮЉв░▒ ВЮ┤в»ИВДђвАю в│ђьЎўьЋювІц.
ВЌгЖИ░Вёю ВцЉВџћьЋю ВаљВЮђ ВъљвДЅВЮђ вїђЖ░ю ВІюВіцьЁю ЖИ░в│ИЖ░њВЮ┤ ьЋўВќђВЃЅВЮ┤ЖИ░ вЋївгИВЌљ Вџ░вдгВЮў 1в╣ёьіИ ьЮЉв░▒ ВЮ┤в»ИВДђВЌљВёювіћ **в░ЮВЮђ ьћйВЁђВЮ┤ 1ВЮ┤ вљўВќ┤ВЋ╝ ьЋювІцвіћ Ж▓ЃВЮ┤вІц**.

![Вџ░ВИА 3Ж░юВЮў ВЮ┤в»ИВДђВЌљВёю в│╝ Вѕў ВъѕвЊ»ВЮ┤, ЖиИ ВѓгВЮ┤ВЮў Gray ВЌєВЮ┤ #000000 Ж▓ђВаЋЖ│╝ #FFFFFF ьЋўВќЉвДїВю╝вАю Ж▓йЖ│ёВёаЖ│╝ ьўЋВ▓┤вЦ╝ вДївЊцВќ┤вѓ╝ Вѕў ВъѕвІц. DemonDeLuxe (Dominique Toussaint), CC BY-SA 3.0, via Wikimedia Commons.](9F37F9.png)

ВаюВЮ╝ ВЎ╝ВфйВЮў ВЮўв»ИВДђвіћ 256вІеЖ│ёВЮў grayscale ВЮ┤в»ИВДђВЮ┤Ж│а,
вѓўвеИВДђ 3Ж░юВЮў ВЮ┤в»ИВДђвіћ Ж░ЂЖИ░ вІцвЦИ ВЋїЖ│авдгВдўВю╝вАю вѓўьЃђвѓИ вфевЁИьЂгвАг ВЮ┤в»ИВДђВЮ┤вІц.
ВЮ┤ ьћёвАюВаЮьіИВЌљВёювіћ ВхюВбЁВаЂВю╝вАю Floyd-Steinberg ВЋїЖ│авдгВдўВЮё ВѓгВџЕьЋювІц.

### ВЮ┤в»ИВДђвЦ╝ вфевЁИьЂгвАгВю╝вАю в│ђьЎўьЋўЖИ░

ВЮ┤в»ИВДђвЦ╝ 1в╣ёьіИ ьЮЉв░▒Вю╝вАю в░ћЖЙИвіћ в░Ев▓ЋВЮђ ЖхЅВъЦьъѕ вІцВќЉьЋювЇ░, ВЮ┤ ьћёвАюВаЮьіИвіћ sRGB ВўЂВЌГвДї ВѓгВџЕьЋа Ж▓ЃВЮ┤ЖИ░ВЌљ CIE 1931 sRGBВЌљ ВаЋВЮўвљю Luminance ЖИ░в░ў в│ђьЎўВЮё ВѓгВџЕьЋювІц.
вІцВЮїВ▓ўвЪ╝ Ж░ёвІеьЋўЖ▓ї ьЉюьўёьЋа Вѕў ВъѕвІц. [В░ИЖ│а](https://en.wikipedia.org/wiki/Grayscale#Converting_colour_to_greyscale)

```python
def grayscale(red: int, green: int, blue: int) -> int:
    return int(0.2126 * red + 0.7152 * green + 0.0722 * blue)
```

ВЌгЖИ░ВЌљВёю red, green, blueвіћ 0-255ВЮў `int`ВЮ┤вІц.
ВЮ┤ ьЋЕВЮ┤ ВъёВЮўВЮў `hex_threshold`вЦ╝ вёўВю╝вЕ┤ ьЋ┤вІ╣ ьћйВЁђВЮў Ж░њВЮё 1вАю ВёцВаЋьЋювІц.
ВЮ┤ ВйћвЊювЦ╝ вфевЊа ьћйВЁђвДѕвІц ВІцьќЅьЋ┤ВБ╝вЕ┤ вљювІц. вг╝вАа ВюёВЮў grayscale ВйћвЊювіћ ВЮ┤вАаВаЂВЮИ вХђвХёВЮё ВЮ┤ьЋ┤ьЋўЖИ░ ВюёьЋеВЮ┤Ж│а,
ВхюВбЁВаЂВю╝вАювіћ ВЋёвъўВЎђ Ж░ЎВЮ┤ `Python PIL`ВЌљ вѓ┤ВъЦвљю ВйћвЊювЦ╝ ВѓгВџЕьЋа Ж▓ЃВЮ┤вІц.
ьЏёВѕаьЋўЖ▓аВДђвДї, ВЮ┤ вЮ╝ВЮ┤вИївЪгвдгвіћ ЖИ░в│ИВаЂВю╝вАю **вћћвЇћвДЂ В▓ўвдг**вЈё ьЋ┤ВцђвІц.

## 2\. вфевЁИьЂгвАг ВЮ┤в»ИВДђвЦ╝ ВъёВЮў ьЂгЖИ░ВЮў ВаљВъљ в░░ВЌ┤вАю в│ђьЎў

Вюё вгИВъЦВЮђ 3Ж░ђВДђ ьїїьіИвАю вѓўвѕї Вѕў ВъѕвІц. РЉа 1в╣ёьіИ вфевЁИьЂгвАг ВЮ┤в»ИВДђвЦ╝ РЉА ВъёВЮў ьЂгЖИ░ВЮў РЉб ВаљВъљ в░░ВЌ┤вАю в│ђьЎўьЋўвіћ ЖИ░Вѕа. 1в▓ѕВЮђ ВЎёвБїьќѕВю╝вІѕ Вџ░Вёа 2в▓ѕвХђьё░ Вѓ┤ьј┤в│┤Въљ.

### PILВЮё ьєхьЋ┤ ВЮ┤в»ИВДђ ьЂгЖИ░ в│ђьЎўьЋўЖИ░

ВЮ┤ ьћёвАюВаЮьіИВЌљВёювіћ ВЮ┤в»ИВДђвЦ╝ PILВЮё ВѓгВџЕьЋ┤ вХѕвЪгВўцЖИ░ вЋївгИВЌљ, вІцВЮї ВйћвЊювЦ╝ ьєхьЋ┤ ВЮ┤в»ИВДђвЦ╝ ВъёВЮў ьЂгЖИ░вАю в│ђЖ▓йьЋа Вѕў ВъѕвІц.

```python
def resize(image: Image.Image, width: int, height: int) -> Image.Image:
    if height == 0:
        height = int(im.height / im.width * width)
    if height % braille_config.height != 0:
        height = int(braille_config.height * (height // braille_config.height))
    if width % braille_config.width != 0:
        width = int(braille_config.width * (width // braille_config.width))
    return image.resize((width, height))
```

ВЮ┤ ьћёвАюВаЮьіИВЌљВёю ВѓгВџЕьЋа ВаљВъљвіћ ьЂгЖИ░Ж░ђ 2 x 3 ьЂгЖИ░ВЮ┤ЖИ░ВЌљ ВЮ┤в»ИВДђВЮў вёѕв╣ё, вєњВЮ┤Ж░ђ ВЌгЖИ░ВЌљ ВЎёв▓йьЋўЖ▓ї вѓўвѕёВќ┤ вќеВќ┤ВДђвЈёвАЮ ьЂгЖИ░вЦ╝ в»ИВёИьЋўЖ▓ї ВА░ВаЋьЋювІц.
