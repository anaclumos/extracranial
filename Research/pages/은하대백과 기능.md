## 25성현이 생각했을 때 있어야 하는 기능

### AI 네이티브 리라우팅

- 포스트 끝에 `.txt`나 `.md`를 붙이면 Plain Text 버전이 나옴
- User-Agent를 감지하고, ChatGPT 등인 경우 이 Plain Text 사이트로 리라우팅

### AI 네이티브 대화 임포트

AI와의 대화 내용이 곧 지식의 탐구 내용으로서 정리되는 UI 형태. 근데 위에 언급한 대화록 UI로서 그냥 될 것 같기도 하고...

### 다단처리

여러 개 문단으로 좌우, 마치

### `Shoutout`, `Quote`, `Quote[]`

이 셋은 서로 다른 개념으로 구분되어야 함. Shoutout은 내가 그냥 강조하고 싶은 내용들, Quote는 단일 인용, `Quote[]`는 인용의 쓰레드 개념으로 구분되어야 함.

### `Collapsible`

`details` `summary`를 쓰지 않고 좀 더 쉬운 문법이 필요함.

### 화이트보드 / 태스크보드

위에서 아래로 읽는 글이 아니라 작은 포스트잇들이 수십개 붙어 있는 형태의 보드 UI
