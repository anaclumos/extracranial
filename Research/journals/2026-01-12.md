## AGENTS

Implemented Obsidian-style graph view using [[Sigma.js]] with @react-sigma/core:

- Full-page interactive knowledge graph at `/graph`
- 3,038 nodes, 7,983 edges visualized with WebGL
- Features: search with autocomplete, zoom/pan controls, hover highlighting, click-to-navigate
- Node sizing based on connection count (degree centrality)
- Color-coded by type: journals (blue), pages (green), coding problems (amber)
- Dark/light mode support via Docusaurus theme integration
- New files: `src/components/graph-view/`, `src/pages/graph/`, `src/utils/graph-data.ts`
