---

The file `strings.xml` is located at `res/values/strings.xml`.

```xml
<resources>  
    <string name="app_name">CS310 Sunghyun Cho (scho1939)</string>  
    <string name="message">Fight On!</string>  
</resources>
```

Then refer the text like

```
android:text="@string/message"
```
