> [React](./../.././docs/pages/React.md) Native is an open-source [UI](./../.././docs/pages/UI%20and%20UX.md) software framework created by [Meta](./../.././docs/pages/Meta%20%28Company%29.md) Platforms, Inc. It is used to develop applications for [Android](./../.././docs/pages/Android.md), [Android](./../.././docs/pages/Android.md) TV, [iOS](./../.././docs/pages/iOS.md), [macOS](./../.././docs/pages/macOS.md), tvOS, Web, Windows, and UWP by enabling developers to use the [React](./../.././docs/pages/React.md) framework and native platform capabilities. It is also being used to develop [virtual reality](./../.././docs/pages/Virtual%20Reality.md) applications at Oculus.
