## [refinedev/refine](https://github.com/refinedev/refine): Build your React-based CRUD applications, without constraints.

It will generate a

- React Platform (Vite, Remix, Next.js)
- UI Framework (Material UI, Ant Design, Chakra UI, Mantine, [Headless](./../.././docs/pages/Headless.md))
- [Backend](./../.././docs/pages/Back-end.md) (REST [API](./../.././docs/pages/API.md), Supabase, Strapi, NestJS, Appwrite, Airtable, Hasura)
- Authentication Provider (Google Auth, Keycloak, Auth0, Supabase, Appwrite, Strapi, Custom Auth, No Auth)
