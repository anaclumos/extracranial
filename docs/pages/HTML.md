> The HyperText Markup [Language](./../.././docs/pages/Language.md) or HTML is the standard markup [language](./../.././docs/pages/Language.md) for documents designed to be displayed in a [web browser](./../.././docs/pages/Web%20Browser.md). It can be assisted by technologies such as [Cascading Style Sheets](./../.././docs/pages/CSS.md) ([CSS](./../.././docs/pages/CSS.md)) and scripting languages such as [JavaScript](./../.././docs/pages/JavaScript.md).
