Open Source Software Accelerationism (oss/acc) suggests a philosophy or movement that advocates for the rapid advancement of technology through open-source software development practices. It is based on [Effective Accelerationism](./../.././docs/pages/%ED%9A%A8%EA%B3%BC%EC%A0%81%20%EA%B0%80%EC%86%8D%EC%A3%BC%EC%9D%98.md).
