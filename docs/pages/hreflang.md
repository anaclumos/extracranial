> The `rel="alternate"` `hreflang="x"` link attribute is an [HTML](./../.././docs/pages/HTML.md) [meta](./../.././docs/pages/Meta.md) element described in RFC 8288. Hreflang specifies the [language](./../.././docs/pages/Language.md) and optional geographic restrictions for a document. [Search engines](./../.././docs/pages/Search%20Engine.md) interpret Hreflang and can be used by web admins to clarify the linguistic and geographical targeting of a website. [hreflang](https://en.wikipedia.org/wiki/Hreflang)

important for [i18n](./../.././docs/pages/Internationalization.md) with [Content Negotiation](./../.././docs/pages/Content%20Negotiation.md)
