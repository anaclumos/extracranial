> Write once, run anywhere (WORA), or sometimes Write once, run everywhere (WORE), was a 1995 slogan created by Sun Microsystems to illustrate the cross-platform benefits of the [Java](./../.././docs/pages/Java.md) language. Ideally, a [Java](./../.././docs/pages/Java.md) programs could be developed on any device, compiled into standard bytecode, and be expected to run on any device equipped with a [Java](./../.././docs/pages/Java.md) virtual machines ([JVM](./../.././docs/pages/JVM.md)). Installing a [JVM](./../.././docs/pages/JVM.md) or [Java](./../.././docs/pages/Java.md) interpreters on chips, apparatus, or software packages became an industry standard practice.
