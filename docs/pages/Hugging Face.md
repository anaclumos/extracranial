> Hugging Face, Inc. is an [American](./../.././docs/pages/American.md) company that develops tools for building applications using [machine learning](./../.././docs/pages/ML.md). It is most notable for its Transformers library built for [natural language processing](./../.././docs/pages/Natural%20Language%20Processing.md) applications and its platform that allows users to share [machine learning](./../.././docs/pages/ML.md) models and datasets.
