[Cloudflare](./../.././docs/pages/Cloudflare.md) Worker is a [serverless](./../.././docs/pages/Serverless.md) execution environment that allows users to create and deploy [JavaScript](./../.././docs/pages/JavaScript.md) or [WebAssembly](./../.././docs/pages/WebAssembly.md) code at the [edge network](./../.././docs/pages/Edge%20Network.md) of [Cloudflare](./../.././docs/pages/Cloudflare.md)'s global network for high-performance, low-latency applications, and dynamic content delivery.
