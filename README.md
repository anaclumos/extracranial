# Extracranial

A personal wiki and digital garden built with Next.js, featuring MDX content rendering with wiki-style linking.

## Getting Started

```bash
bun install
bun run dev
```

## Building For Production

```bash
bun run build
```

## Linting & Formatting

This project uses [Ultracite](https://github.com/anthropics/ultracite) (Biome preset) for linting and formatting:

```bash
bun x ultracite fix    # Format and fix lint issues
bun x ultracite check  # Check for issues without fixing
```
